# ⌨️ Basic Keylogger

## 📌 Project Overview

This project was developed as **Task 04** of my **Skill High Cyber Security Internship**.

The Basic Keylogger is a simple Python program that records keyboard keystrokes and stores them in a text file. This project is created purely for educational purposes to understand how keyboard event logging works and its relevance in cybersecurity.

---

## 🚀 Features

- Records keyboard keystrokes.
- Logs the pressed keys into a text file.
- Demonstrates keyboard event handling in Python.
- Helps understand the basics of keylogging.

---

## 🛠️ Technologies Used

- Python
- pynput Library

---

## 📚 What I Learned

- Learned how keyboard events can be captured using Python.
- Explored the `pynput` library for monitoring keyboard inputs.
- Understood how keystrokes can be written and stored in a text file.
- Learned the importance of keyloggers in cybersecurity awareness and ethical security testing.
- Gained practical experience in building a basic event-driven Python application.

This project was created **only for educational and learning purposes** as part of my cybersecurity internship. It is intended to understand how keylogging works and to promote awareness of cybersecurity concepts. It should only be used in authorized and ethical environments.
